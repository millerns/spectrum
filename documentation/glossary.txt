/////////////////////
Glossary for Spectrum
/////////////////////
Item - A discrete thing that can be included in a set or list. It is generated by a user.
Set - An unsorted collection of unique items. It is generated by a user who provides the items.
List - A sorted collection of unique items. It is generated by the application from a provided set following favoriting.
Decision - The process in which the user decides between one of two items from a given set. The favored item rises in the eventually generated list, while the other item falls.
Favoriting - The collection of decisions made on a single set. This process generates the list for that set.
